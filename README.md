# 🎟️ Eventos MVP

Sistema web para gerenciamento e visualização de eventos, desenvolvido em React como um MVP (Minimum Viable Product).

A aplicação permite visualizar eventos disponíveis, acessar detalhes e simular uma inscrição.
O projeto foi criado com foco em aprendizado de desenvolvimento web moderno, estrutura de componentes e organização de código.

Aplicação web desenvolvida em **React** para gerenciamento e visualização de eventos.  
O projeto permite listar eventos, visualizar detalhes e simular um fluxo de inscrição.

Este projeto foi desenvolvido como **MVP (Minimum Viable Product)** para fins de aprendizado e prototipação de um sistema de gerenciamento de eventos.

# 🧠 Objetivo do projeto

- Praticar desenvolvimento com React
-Trabalhar com componentização
-Organizar páginas e rotas
-Simular um sistema real de eventos
-Servir como base para evolução com backend e banco de dados

# ⚙️ Como rodar o projeto

### 1️⃣ Clonar o repositório
git clone https://github.com/jofavaretto/eventos-mvp.git


### 2️⃣ Entrar na pasta do projeto
cd eventos-mvp

### 3️⃣ Instalar as dependências
npm install

### 4️⃣ Rodar o projeto
npm run dev
